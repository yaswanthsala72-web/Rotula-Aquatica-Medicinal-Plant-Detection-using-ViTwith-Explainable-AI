# Rotula Aquatica Medicinal Plant Detection using Vision Transformer (ViT) with Explainable AI

## Overview

This project presents an AI-powered medicinal plant identification system that leverages Vision Transformer (ViT) technology and Explainable AI (XAI) techniques to accurately recognize Rotula Aquatica, a rare medicinal plant known for its therapeutic properties. The system combines deep learning, computer vision, and interpretable AI to provide reliable plant classification and visual explanations for predictions.

## Objectives

* Automate medicinal plant identification using AI.
* Improve classification accuracy through Vision Transformers.
* Provide transparent and interpretable predictions using Explainable AI.
* Support biodiversity conservation and botanical research.

## Features

* Medicinal plant image classification.
* Vision Transformer (ViT) based feature extraction.
* Explainable AI visualization using Grad-CAM and SHAP.
* High-accuracy plant recognition.
* User-friendly interface for image upload and prediction.
* Educational information about identified medicinal plants.

## Technologies Used

* Python
* TensorFlow / Keras
* OpenCV
* Vision Transformer (ViT)
* Grad-CAM
* SHAP
* NumPy
* Streamlit / Flask

## Methodology

1. Image Acquisition and Upload
2. Image Preprocessing and Augmentation
3. Vision Transformer Feature Extraction
4. Plant Classification
5. Explainable AI Visualization
6. Result Display and Interpretation

## Results

* Achieved 97.5% classification accuracy for medicinal plant recognition.
* Outperformed traditional CNN-based models.
* Reduced misclassification errors through transformer-based learning.
* Generated interpretable visual explanations for model predictions.
